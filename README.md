##나이키 로고
<img width="735" height="309" alt="logo" src="https://github.com/user-attachments/assets/fe4ee687-e9d7-4ab3-a091-8b9b8b5847bf" />

나이키랑 잘 어울리는 검정색 배경화면 ♥ 넣어서 느낌있게 했습니다.
나이키 특유의 깔끔하고 힙한 감성을 살리기 위해 여백(Margin/Padding)과 폰트 크기 조절에 가장 공을 들였습니다.
카테고리에 마우스를 올렸을 때 통통한 손모양 마우스로 바뀌도록 변경하였습니다.
현재는 메인 페이지만 구현되어 있지만, 추후 상품 상세 페이지와 개별 장바구니 담기 기능까지 자바스크립트로 구현해 볼 예정입니다.
